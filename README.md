# Profile

A simple profile app to display biodatas and skills. This is a week one project from compilecamp bootcamp

## Features

- 📷 Profile Picture
- 📄 Name, Bio, and Contact Information
- 🔗 Social Media Links
- 🧑‍💻 Skills and Experience Section
- 📞 Call / Email Buttons
